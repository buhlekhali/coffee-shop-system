# Coffee Shop Order System

## Project Description
A C# desktop application for managing coffee shop orders, with features:
- Menu customization
- Order processing
- Receipt generation
- Sales reporting

## Team Members
- Buhlebuzile Khali 
- Tiyane Sibuya

## Technologies
- C# .NET Framework
- Windows Forms
- SQLite Database

## Setup
1. Clone this repository
2. Open `CoffeeShop.sln` in Visual Studio
3. Build and run (F5)
